
# Blueprint: Prezentace Firebase pro EdTech

## Přehled

Cílem tohoto projektu je vytvořit jednoduchou, moderní a vizuálně přitažlivou jednostránkovou webovou prezentaci, která představí platformu Firebase a její relevanci pro sektor vzdělávacích technologií (EdTech).

## Struktura a Design

### Struktura Projektu
- **`index.html`**: Hlavní HTML soubor obsahující strukturu stránky.
- **`style.css`**: Soubor pro stylování, využívající moderní CSS vlastnosti (proměnné, flexbox, grid).
- **`main.js`**: JavaScript soubor pro budoucí interaktivitu (např. animace, web components).
- **`blueprint.md`**: Tento soubor.

### Rozložení Stránky (Layout)
1.  **Hlavička (Hero sekce)**: Velký, poutavý nadpis "Firebase pro EdTech" s krátkým podtitulem.
2.  **Sekce "Co je Firebase?"**: Stručné vysvětlení platformy.
3.  **Sekce "Klíčové vlastnosti"**: Přehled nejdůležitějších funkcí Firebase (Authentication, Firestore, Storage, Hosting) formou karet.
4.  **Sekce "Proč Firebase v EdTechu?"**: Výhody použití Firebase pro vývoj vzdělávacích aplikací.
5.  **Patička**: Základní informace.

### Design
- **Barevná paleta**: Inspirována logem Firebase (žlutá, oranžová) v kombinaci s neutrální šedou a bílou.
- **Typografie**: Moderní, bezpatkové písmo (např. Roboto, Open Sans) pro dobrou čitelnost.
- **Ikonografie**: Využití jednoduchých ikon pro vizuální doplnění klíčových vlastností.
- **Interaktivita**: Jemné animace při scrollování nebo najetí myší.
