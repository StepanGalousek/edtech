
# Blueprint: Vánoční Prezentace Firebase pro EdTech

## Přehled

Cílem tohoto projektu je vytvořit jednoduchou, moderní a vizuálně přitažlivou jednostránkovou webovou prezentaci s vánoční tématikou, která představí platformu Firebase a její relevanci pro sektor vzdělávacích technologií (EdTech).

## Struktura a Design

### Struktura Projektu
- **`index.html`**: Hlavní HTML soubor obsahující strukturu stránky.
- **`style.css`**: Soubor pro stylování, využívající moderní CSS vlastnosti pro vánoční design.
- **`main.js`**: JavaScript soubor pro interaktivitu, včetně efektu padajícího sněhu.
- **`blueprint.md`**: Tento soubor.

### Rozložení Stránky (Layout)
1.  **Hlavička (Hero sekce)**: Velký, poutavý nadpis "Firebase pro EdTech" s vánočním pozadím a podtitulem.
2.  **Sekce "Co je Firebase?"**: Stručné vysvětlení platformy.
3.  **Sekce "Klíčové vlastnosti"**: Přehled funkcí Firebase formou karet ve vánočním designu.
4.  **Sekce "Proč Firebase v EdTechu?"**: Výhody použití Firebase.
5.  **Patička**: Základní informace.

### Vánoční Design
- **Barevná paleta**: Červená, zelená, zlatá, bílá a tmavě šedá.
- **Typografie**: Moderní, bezpatkové písmo.
- **Ikonografie**: Využití jednoduchých ikon.
- **Interaktivita**: Efekt padajícího sněhu na pozadí.
